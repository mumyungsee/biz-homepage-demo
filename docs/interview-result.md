# 인터뷰 결과: 산책길카페

## STEP 0. 현황 진단
- 현재 온라인 노출: 인스타 비정기 운영(메뉴 사진 위주, 팔로워 적음), 네이버 플레이스 미등록 추정
- 입지 특성: 골목 안쪽, 처음 오는 손님은 찾기 어려움 → 단골형 입지
- 전략 가설: **지도 검색형(네이버 플레이스 강화)** 우선
  - "찾아오기 어렵다"가 핵심 페인포인트 → 사이트 첫 화면에서도 "오시는 길/지도"를 상단 우선 배치
  - 인스타는 보조 채널로 (DM 연락 동선은 유지)

## STEP 1. 업의 정의
> 산책길카페 — 동네 사랑방 같은 카페. 직접 내린 핸드드립 한 잔과,
> 한 번 먹으면 또 찾게 되는 수제 치즈케이크

## STEP 2. 고객의 언어
- "와이파이 비번 뭐예요?" → 첫 화면 또는 안내 영역에 와이파이 정보 노출
- "포장 되나요?" → 포장 가능 안내
- "강아지 데리고 들어가도 돼요?" → 반려동물 동반 가능 안내
- "원두 뭐 쓰세요?" → 메뉴 설명에 원두 정보 포함
- "사장님이 친절하다" / "치즈케이크 진하다" → 후기 섹션 또는 첫 화면 문구 후보

## STEP 3. 첫 화면 구성안 (STEP0 진단 반영 — 지도 검색형 우선)
1. **상단**: 메인 문구 "동네 사랑방 같은 카페, 산책길카페" + 오시는 길/지도 (가장 먼저 보이게)
2. 대표 사진/콘텐츠 3개:
   - 핸드드립 내리는 모습
   - 치즈케이크 클로즈업
   - 가게 안 분위기 (손님 + 강아지)
3. 안내 영역: 포장 가능, 반려동물 동반 가능, 와이파이 제공
4. 연락 동선: 지도(길찾기) 우선, 인스타그램 DM 보조
