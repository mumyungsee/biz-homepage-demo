# 참고자료: 소상공인 홈페이지 구축 교육 (박종선님 제공)

> 원본 PDF는 이 폴더에 직접 복사해두면 좋음 (소상공인_홈페이지_구축_교육.pdf)

## 핵심 철학
- 기술을 가르치는 교육이 아니라, 사장님의 장사를 표현하는 교육
- 수료 = 내 가게의 영업사원 1명 완성
- "좋은 홈페이지는 잘 만든 사이트가 아니라, 내 장사가 잘 보이는 사이트"

## 3개 축 + 1개 관점
1. 도메인 — 내 가게의 주소
2. 서버 — 내 가게의 땅
3. 특화 홈페이지 — 내 업의 표현
- 관통 관점: 마케팅 ("고객이 어떻게 찾아오고, 무엇을 보고, 왜 연락하게 되는가")

## 핵심 워크숍 — 내 업(業)을 한 화면에 담기 (이번 데모에 바로 적용할 부분)
- STEP 1. 업의 정의: "나는 무엇을 파는 사람인가"를 한 문장으로
  (예: 빵집이 아니라 '아침을 여는 빵', 수리점이 아니라 '하루 만에 고치는 집')
- STEP 2. 고객의 언어: 손님이 실제로 검색/질문하는 말 그대로 적기
  → 첫 화면 문구와 메뉴 이름이 됨
- STEP 3. 화면 구성: 한 문장 + 대표 사진 3장 + 연락 버튼으로 첫 화면 스케치

## 기술 용어 → 마케팅 언어 변환표
| 기술의 언어 | 마케팅의 언어 |
|---|---|
| 반응형 웹을 적용합니다 | 손님 10명 중 8명은 폰으로 봅니다. 폰에서 안 예쁘면 없는 가게입니다 |
| 메타태그와 SEO를 설정합니다 | 네이버·구글에서 검색했을 때 우리 가게가 나오게 만듭니다 |
| 문의 폼을 추가합니다 | 밤 11시에 본 손님도 다음 날 연락할 수 있게 합니다 |
| 갤러리 페이지를 만듭니다 | 말로 설명 못 하는 솜씨를 사진 10장으로 증명합니다 |

## 수강생 역량 그룹 (11월 강의 설계 시 참고)
- 입문 그룹: 마우스·파일 관리부터 어색함 → 따라하기 시트 + 화면 그대로 실습
- 중간 그룹: SNS는 하지만 홈페이지 처음 → 내 콘텐츠 옮기기 실습 집중
- 실전 그룹: 사이트 경험 있음 → 심화 미션 + 동료 멘토

## 8차시 커리큘럼 예시 (11월 강의 설계 시 참고)
| 차시 | 주제 | 결과물 |
|---|---|---|
| 1 | 오리엔테이션 + 내 업 정의 워크숍 | 내 업 한 문장 + 첫 화면 스케치 |
| 2 | 도메인 이해와 구매 | 내 가게 도메인 확보 |
| 3 | 호스팅 선택과 연결 | 도메인-서버 연결 완료 |
| 4 | 홈페이지 골격 만들기 | 메인 + 소개 페이지 |
| 5 | 내 업 표현하기 (사진·문구) | 상품/메뉴 페이지 완성 |
| 6 | 고객 연결 (전화·지도·문의) | 연락 동선 완성 |
| 7 | 검색 노출과 마케팅 기초 | 네이버·구글 등록 |
| 8 | 발표: 내 홈페이지 소개 | 공개 가능한 홈페이지 1개 |

## 운영 팁 (11월 강의 설계 시 참고)
- 따라하기 시트 (화면 캡처 포함 1장)
- 동료 멘토 (실전 그룹이 도우미)
- 차시별 완성 기준 명확히
- 자기 가게 소재만 사용 (예제 데이터 금지)
