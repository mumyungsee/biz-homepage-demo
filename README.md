11월 소상공인 대상 강의 의뢰를 계기로 시작한 프로젝트입니다. 가상의 카페 "산책길카페"를 예시로, AI와 대화하며 홈페이지를 직접 만드는 과정을 데모로 만들었습니다.

컨설팅 에이전트와 페르소나 에이전트가 대화하며 현황 진단 → 업의 정의 → 고객의 언어 → 화면 구성을 끌어내는 워크숍 구조 자체가 "AI로 고객 니즈를 끌어내는 법" 강의 콘텐츠가 됩니다.
